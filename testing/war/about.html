ABOUT US

The main idea of this project is to build a mobile application for the college, which involves smarter way of alerting the people through notifications at different levels. It is a 3-level notification system within college i.e., for college, department and class. It requires building an admin page where only authorized persons can update information to be sent as notification to specific set of people based on the reachability he/she have. On the whole this project emphasizes on how a mobile application can act as an e-notice board..
